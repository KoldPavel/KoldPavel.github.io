DirectShape из точек

Создаёт КЭ-схему в Revit из файлов CSV/XLSX, формируя DirectShape и параметры проекта для дальнейшего анализа.

Что делает плагин

Инструмент строит КЭ схему в модели Revit, аналогичную расчётной модели ЛИРЫ, на основе таблиц CSV или XLSX. Он создаёт DirectShape в категории «Антураж», добавляет параметры проекта и записывает значения армирования для каждого конечного элемента. Перед запуском таблицы экспортируются из ЛИРЫ, после чего весь объём данных переносится в Revit для дальнейшей работы.

Ключевые возможности

  • Импорт таблиц CSV/XLSX и создание DirectShape, соответствующих расчётной модели.
  • Запись параметров армирования в проект для последующих проверок и визуализации.
  • Полная синхронизация данных из ЛИРЫ с моделью Revit без ручного ввода.
