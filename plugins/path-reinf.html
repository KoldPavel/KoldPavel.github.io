Дополнительное армирование

Расставляет дополнительные зоны армирования по изополям и перекрашивает пересечения с существующими DirectShape.

Что делает плагин

Плагин использует изополя из ЛИРЫ для создания дополнительных зон армирования. Он автоматически определяет места перекрытия с существующими DirectShape, добавляет необходимые стержни и может перекрашивать элементы, где выполнено усиление, для наглядного контроля объёмов армирования.

Ключевые возможности

  • Построение дополнительных участков армирования по импортированным изополям.
  • Перекрашивание DirectShape, которые пересекаются с новыми зонами армирования.
  • Контроль корректности армирования через визуальную проверку в модели.
