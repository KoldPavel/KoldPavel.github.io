Дополнительное армирование

Расстановка зон армирования по изополям из ЛИРЫ

Армирование

Плагин расставляет дополнительные зоны армирования по изополям, полученным из ЛИРЫ, и позволяет перекрасить DirectShape, которые перекрываются дополнительным армированием.

Ключевые возможности

  • Использование данных из ЛИРЫ для расстановки дополнительных зон армирования.
  • Быстрая окраска DirectShape в местах пересечения с дополнительным армированием.
  • Контроль соответствия расчётных изополей и арматурных зон.
