Армирование

Плагин расставляет дополнительные зоны армирования по изополям, полученным из ЛИРЫ, и позволяет перекрасить DirectShape, которые перекрываются дополнительным армированием.

Ключевые возможности

  • Использование данных из ЛИРЫ для расстановки дополнительных зон армирования с учётом коэффициента использования арматуры.
  • Быстрая окраска DirectShape в местах покрытия дополнительным армированием.
