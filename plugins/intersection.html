Назначить хосты DS

Подготавливает DirectShape для дальнейшего армирования, сопоставляя элементы с их исходными конструкциями.

Что делает плагин

Инструмент сопоставляет ID DirectShape с ID конструкций, которым они принадлежат. Это обеспечивает корректные связи перед запуском плагина «Дополнительное армирование», чтобы дополнительные зоны армирования автоматически находили нужные хосты.

Ключевые возможности

  • Поиск пересечений между DirectShape и элементами КР в модели.
  • Сохранение связей ID для последующих расчётов и окрашивания.
  • Подготовка модели к работе с дополнительным армированием без ручных сопоставлений.
