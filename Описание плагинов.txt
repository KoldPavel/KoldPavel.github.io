Создание видов
Создаёт разрезы с заданными шаблонами и границами обрезки по выбранным конструкциям (стены, колонны, балки). Также можно создать листы и спецификации и вынести виды на листы.

Оформление видов
Обрабатывает выбранные разрезы. Размещает размерные цепочки, марки арматуры, линии обрыва, высотные отметки. 

Назначить хосты DS
Служит для подготовки DirectShape к использованию в плагине "Дополнительное армирование". Соотносит id DS с id конструкции, к которой DS принадлежит.

Просмотр результатов
Позволяет раскрасить DirectShape в соответствии со значениями площади армирования, как в ЛИРА САПР. Раскраска идет по настроенной цветовой шкале.

Дополнительное армирование
Расставляет дополнительные зоны армирования по изополям, полученным из ЛИРЫ. Также можно перекрасить DirectShape, которые перекрываются доп армированием.

DirectShape из точек
Строит КЭ схему в модели Revit аналогичную схеме в ЛИРЕ по таблицам CSV или XLSX. Также создаёт параметры проекта для категории Антураж, в которые записывает значения армирования каждого КЭ. КЭ создаются в виде DirectShape в категории Антураж. Предварительно таблицы нужно экспортировать из ЛИРЫ.
